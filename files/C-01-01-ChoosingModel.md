# 模型选择

> 根据您的研究目标选择合适的结构预测模型

HelixFold 平台提供两种结构预测模型 HelixFold3 与 HelixFold-S1，适用于不同的研究场景。在开始结构预测之前，选择正确的模型是确保预测成功的关键第一步。

## 模型快速对比

| 特性 | HelixFold3 | HelixFold-S1 |
|------|------------|--------------|
| **模型定位** | 通用结构预测模型 | 多链复合物增强版 |
| **适用场景** | 蛋白-蛋白、蛋白-小分子、蛋白-RNA 等 | 抗原-抗体复合物（推荐）、高精度链间构象 |
| **单链支持** | ✅ 支持 | ❌ 不支持 |
| **化学修饰** | ✅ 支持 | ❌ 暂不支持 |
| **参考结构引导** | ✅ 支持 | ✅ 支持 |
| **相互作用约束** | ✅ 支持 | ✅ 支持 |

## 适用场景

### HelixFold3 适用场景
HelixFold3 是通用模型，适合在绝大多数场景中作为默认模型使用，这些场景包括：

- 蛋白-蛋白、蛋白-小分子、蛋白-RNA 等复合物预测
- 需要自定义化学修饰的任务
- 单链蛋白质结构预测
- 需要快速获得结果的常规预测任务

### HelixFold-S1 适用场景
HelixFold-S1 是增强版本的模型，更适合在链间交互显著的体系中使用：

- **抗原-抗体复合物**（推荐使用）
- 多链蛋白-蛋白相互作用界面预测
- 需要高精度链间构象的研究
- 关注蛋白-蛋白界面的详细结构

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| S1 不支持单链任务 | HelixFold-S1 需要至少两条链 |
| S1 暂不支持化学修饰 | 如需使用化学修饰，请选择 HelixFold3 |
| S1 不支持 Recycle | 模型设计差异，不影响预测质量 |
