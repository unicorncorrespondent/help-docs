# 点券系统

> 了解点券使用与获取规则，充分利用平台优惠

点券是 PaddleHelix 平台提供的虚拟代金券，可用于抵扣平台任务费用。本页面汇总点券的获取方式、类型说明、使用规则与查询方法，帮助您清晰了解点券机制并合理使用。


## 什么是点券？

点券是平台发放的虚拟优惠凭证，用于抵扣部分任务费用，以降低使用成本。1 点券 = 1 元人民币，可按 1:1 抵扣平台任务费用（在规则允许范围内）。

点券可能存在以下类型：

- **全局点券**：可抵扣任意模块费用  
- **单模块点券**：仅限指定模块使用  
- **活动点券**：可能附带有效期或使用条件  

不同类型点券的适用范围与有效期以账户内显示为准。



## 获取点券

### 免费获取

用户可通过完成平台任务自动获得点券：

| 方式 | 点券数量 | 条件 |
|------|----------|------|
| 新用户注册 | 见活动规则 | 新账号注册 |
| 完成用户调研 | 见活动规则 | 参与问卷或反馈 |
| 邀请好友 | 见活动规则 | 成功邀请注册 |
| 升级高级版 | 见活动规则 | 完成实名及开通CHPC |
| 订阅平台动态 | 见活动规则 | 订阅通知或资讯 |
| 日常活跃 | 随机 | 每日抽奖等活动 |
| 月度活动 | 以活动页为准 | 每月任务或活动 |
| 体验反馈 | 5–10 点等 | 提交体验评价 |

> 实际奖励数量以活动页面或系统显示为准。

### 活动赠送

平台在以下场景可能发放额外点券：

| 活动类型 | 点券数量 |
|----------|----------|
| 节假日活动 | 不定 |
| 周年庆活动 | 大额发放 |
| 用户回馈 | 定期发放 |
| 专项推广活动 | 以活动说明为准 |



## 点券类型与有效期

不同来源的点券可能存在不同有效期，请在账户中查看具体到期时间。

| 类型 | 有效期说明 |
|------|------------|
| 新用户点券 | 一般为注册后固定期限 |
| 活动点券 | 以活动规则为准 |
| 任务奖励点券 | 可能存在月度或阶段有效期 |
| 购买或长期点券 | 通常为较长有效期 |

> 部分点券可能仅当月有效或限期使用，请及时使用避免过期。


## 使用规则

### 抵扣比例与方式

- 点券按 **1:1** 抵扣人民币费用  
- 系统将优先使用可用点券抵扣任务费用  
- 高级版账号支持点券与账户余额混合支付  
- 点券余额不足时，将自动从账户余额补扣  

### 抵扣优先顺序

当账户中存在多张点券时，系统按以下顺序自动抵扣：

1. 点券类型：优先使用单模块点券，其次全局点券  
2. 有效期：优先使用即将到期的点券  
3. 发放时间：优先使用较早发放的点券  

### 使用限制

| 规则 | 说明 |
|------|------|
| 是否可提现 | 不可提现，仅限平台使用 |
| 是否可转让 | 不可转让，仅限本人账号 |
| 是否可退款退回 | 已使用点券退款后不退回 |
| 是否可与学术优惠叠加 | 不可叠加使用 |
| 普通账号限制 | 仅可用于基础功能体验 |

如需使用学术优惠，请在提交任务前关闭点券抵扣。


## 查看点券

### 查看方式

1. 登录 PaddleHelix 平台  
2. 进入「我的账户」或「账户中心」  
3. 打开点券或账户总览页面  
4. 查看点券余额与有效期  

### 可查看内容

| 项目 | 说明 |
|------|------|
| 点券余额 | 当前可用点券 |
| 有效期 | 每张点券到期时间 |
| 获取记录 | 点券来源记录 |
| 使用记录 | 抵扣记录 |
| 过期记录 | 已失效点券 |

> 相同来源且有效期一致的点券可能合并显示。

## 使用点券

### 自动抵扣（默认）

提交任务时系统将自动计算并使用可用点券：

1. 选择功能并创建任务  
2. 确认订单金额  
3. 系统自动抵扣可用点券  

### 手动调整抵扣

如需调整抵扣方式：

1. 在订单页面进入「订单详情」  
2. 查看点券抵扣情况  
3. 可选择关闭点券或使用其他优惠方式

## 常见问题

### Q1: 点券可以提现吗？
不可以。点券仅用于平台任务费用抵扣。

### Q2: 点券可以转让或共享吗？
不可以，仅限当前账号使用。

### Q3: 点券过期后能恢复吗？
不能。过期后将自动失效。

### Q4: 退款后点券会退回吗？
不会。已使用点券在退款后不予退回。
