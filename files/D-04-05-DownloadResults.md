# 下载结果

> 导出分子设计结果用于后续分析



## 下载方式

### 整体下载

1. 在结果页面点击「下载」按钮
2. 系统将打包所有结果文件

![Download Button](https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA7KwXUJ0aZJPakAmZmBwA9y.png)


### 文件列表

下载结果包含以下文件：

| 文件类型 | 格式 | 说明 |
|----------|------|------|
| 口袋配置文件 | - | 第一个蛋白的口袋坐标及大小 |
| 口袋配置文件 | - | 第二个蛋白的口袋（如有） |
| 结果表格 | .csv | 分子生成结果数据 |
| 结果文档 | .pdf | 分子生成报告 |



![Download Content](https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA7uc-gdTXlLrZvpZLh3bqDO.png)
