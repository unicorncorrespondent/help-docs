# LinearDesign 参数配置

> 设置 Beam Size、平衡因子 λ 与高级约束

在进行 LinearDesign 任务提交前，合理的参数配置可以帮助您在“稳定性（MFE）”“翻译效率（CAI）”和“计算开销”之间取得更合适的平衡。

## 核心参数

关于核心参数如何设置的具体内容，可以参见FAQ。

### Beam Size（集束大小）

Beam Size 决定搜索过程中保留的候选路径数量：

| Beam Size | 特点 | 适用场景 |
|-----------|------|----------|
| 50-100 | 速度快，结果多样性较高 | 快速初筛、大批量任务 |
| ≥N（接近序列长度） | 更接近全局最优 | 需要更高质量解 |
| infinity | 全局搜索，耗时最高 | 最终确认或重点序列 |


### 平衡因子 λ（Lambda）

λ 用于控制优化侧重：

| λ 区间 | 优化侧重 | 典型场景 |
|--------|----------|----------|
| 0 | 仅优化 MFE | 稳定性优先 |
| 0-2 | 偏重稳定性 | 疫苗设计等场景 |
| 2-4 | 稳定性与表达平衡 | 常规综合优化 |
| ≥4 | 偏重 CAI | 追求高表达 |
| ≥50 | 强烈偏重 CAI | 特殊高表达需求 |

## 高级参数（Advance 版本）

### 茎环限制（S1 / S2）

用于限制较长双链结构，常用于控制免疫原性风险：

| 参数 | 含义 | 常见参考值 |
|------|------|------------|
| S1 | 连续双链区域最大长度 | 33 nt |
| S2 | 含 bulge/loop 双链区域最大长度 | 15 nt |

### 5' 端 leader 区域结构优化

可设置 5' 端结构约束长度（常见 0-30 nt），用于减少翻译起始区域结构阻碍。

### 指定 UTR / 排除酶切位点 / 自定义密码子表

Advance 版本支持更多工程化约束：

- 指定 5'UTR / 3'UTR，按既定调控背景进行 CDS 设计
- 排除指定核酸内切酶位点，适配后续工艺要求
- 上传自定义密码子表，适配特定物种或表达系统

## 版本支持对照

| 参数能力 | Basic | Plus | Advance |
|----------|-------|------|---------|
| Beam Size | 50-100 | ≥50 或 infinity | 支持多值 |
| λ | - | 0 或 ≥50 | 支持多值 |
| 茎环限制 | - | 开关设置 | 自定义 S1/S2 |
| 5'端结构优化 | - | 开关设置 | 自定义长度 |
| UTR 指定 | - | - | 支持 |
| 自定义密码子表 | - | - | 支持 |
| 排除酶切位点 | - | - | 支持 |
