# 抗体设计 FAQ

> 常见问题解答


## 基础问题

### Q1: 什么情况下应该使用抗体设计功能？

**抗体设计**适用于：

- 优化已有抗体的亲和力
- 进行抗体人源化改造
- 设计针对特定抗原的新抗体序列
- CDR 区域亲和力成熟

### Q2: 抗体设计与小蛋白设计有什么区别？

| 特性 | 抗体设计 | 小蛋白设计 |
|------|----------|------------|
| 输入 | 抗原-抗体复合物结构 | 靶蛋白结构 |
| 设计目标 | CDR 区域序列优化 | 从头设计新 Binder |
| 输出 | 1000 条候选序列 | 1000 个候选 Binder |
| 适用场景 | 已知抗体优化 | 全新 binder 设计 |

### Q3: 支持纳米抗体设计吗？

支持。纳米抗体（单链 VHH）无需配对链，设计时选择目标纳米抗体链即可。

在了解基础问题后，以下是关于输入操作的常见问题。

## 输入问题

### Q4: 上传结构有什么要求？

- 需要包含**至少一条抗原链和一条抗体重链**
- 格式：`.cif` 格式
- CDR 区域必须完整
- 建议上传抗原-抗体复合物结构

### Q5: 为什么 CDR 区域必须完整？

CDR 是抗体与抗原直接接触的关键区域。如果 CDR 缺失或损坏：

- 系统无法识别结合位点
- 无法进行有效的序列设计
- 设计结果将不可靠

### Q6: 抗原链有缺失怎么办？

系统会**自动尝试补全**：

- 缺失较少（≤10 aa）：通常可以成功补全
- 缺失较多：补全可能失败，影响设计效果
- 建议：尽可能使用完整的抗原结构

### Q7: Fv 区域有缺失怎么办？

| 缺失数量 | 处理方式 | 结果 |
|----------|----------|------|
| ≤ 2 aa | 系统尝试自动修复 | 可继续设计 |
| > 2 aa | 无法进行设计 | 任务失败 |

在了解输入问题后，以下是关于设计参数的常见问题。

## 设计参数问题

### Q8: 可以只设计 CDR 区域吗？

可以。选择**"指定 Fv 区域"**模式，输入 CDR 残基编号即可：

```
示例：CDR-H3 设计：95-110
```

### Q9: 什么是全链设计？

全链设计对**整个抗体链**（包括 Fv 和 Fc）进行序列优化：

- 适合需要全面改造抗体的场景
- 需要 Fc 区域完整（缺失 ≤2 aa）
- 设计结果包含完整抗体序列

### Q10: 链长度限制是多少？

**总长度 ≤ 1000** 个氨基酸：

```
总长度 = 抗原链长度 + 设计链长度 + 配对链长度（如有）
```

在了解设计参数问题后，以下是关于结果解读的常见问题。

## 结果问题

### Q11: 预测结合能为正值正常吗？

**不正常**。正值 ΔG 表示预测的结合不稳定：

- 可能原因：界面设计不合理、构象预测不可靠
- 建议：筛选 ΔG < 0 的结果，优先选择 ΔG < -5 kcal/mol

### Q12: pLDDT 很低怎么办？

pLDDT < 50 表示该区域结构预测不可靠：

1. 检查是否为 CDR 区域
2. 查看是否是柔性区域（末端-loop）
3. 考虑使用其他候选结果

### Q13: Wild type 和 Designed 的区别？

| 类型 | 说明 |
|------|------|
| **Wild type** | 原始输入的抗体序列 |
| **Designed** | 系统设计的新序列 |

建议重点关注 Designed 类型的结果。

在了解结果问题后，以下是关于技术实现的常见问题。

## 技术问题

### Q14: Fc 区域会被移除吗？

仅当以下情况发生：

- Fc 区域缺失氨基酸 **> 2 个**
- 无法进行全链设计
- 系统自动移除 Fc 后继续 Fv 区域设计

### Q15: 可以同时设计重链和轻链吗？

可以。选择目标设计链时选择需要设计的链，系统会同时进行设计。
