**使用 PaddleHelix 平台的常见问题解答**

在使用 PaddleHelix 平台的过程中，您可能会遇到各种关于账号管理、功能使用、计费问题或技术支持方面的疑问。本页面汇集了用户最常询问的问题和答案，涵盖账号登录、功能使用、计费价格、数据隐私等多个方面。无论您是刚开始探索平台的新用户，还是希望深入了解高级功能的资深用户，都可以在这里找到有帮助的信息。

## 账号与登录

### Q1: 如何注册账号？

PaddleHelix 使用百度账号体系：

1. 访问 [PaddleHelix 官网](https://paddlehelix.baidu.com)
2. 点击右上角「登录」
3. 使用百度账号登录或注册新账号

### Q2: 支持哪些登录方式？

| 登录方式 | 说明 |
|----------|------|
| 百度账号 | 主要登录方式 |
| GitHub | 适用于海外用户 |

### Q3: 忘记密码怎么办？

通过百度账号的密码找回流程重置密码。

## 功能使用

### Q4: 免费额度用完了怎么办？

| 方案 | 说明 |
|------|------|
| 购买套餐 | 选择适合的付费套餐 |
| 领取优惠 | 关注活动获取免费额度 |
| 商业合作 | 联系客服获取定制方案 |

### Q5: 预测结果在哪里查看？

1. 登录 PaddleHelix 平台
2. 进入「我的任务」页面
3. 查看历史预测任务和结果

### Q6: 如何导出预测结果？

每个模块的预测结果都可以导出：

- **HelixADMET**：CSV 格式表格
- **HelixVS**：PDB 文件和 CSV 打分表
- **LinearDesign**：TXT 格式序列

## 数据与隐私

### Q7: 上传的数据会被保存吗？

- **临时处理**：数据仅在计算期间使用
- **结果保存**：预测结果可保存至账户
- **隐私保护**：不会将您的数据用于其他用途

### Q8: 如何删除历史任务？

1. 进入「我的任务」页面
2. 选择要删除的任务
3. 点击「删除」按钮

## 技术问题

### Q9: 预测失败了怎么办？

1. 检查输入数据格式是否正确
2. 查看错误提示信息
3. 参考各模块的使用说明
4. 必要时联系技术支持

### Q10: 计算时间过长怎么办？

| 情况 | 建议 |
|------|------|
| 分子数量多 | 等待或分批处理 |
| 网络问题 | 检查网络连接 |
| 系统繁忙 | 稍后重试 |
