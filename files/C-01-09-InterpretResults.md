> 理解结构预测结果并导出分析

正确解读预测结果是整个流程中至关重要的一步。HelixFold 提供了多维度的置信度评估指标，帮助您判断预测结构的可靠程度。完成评估后，您可以将结果导出到本地进行进一步分析。

## 关键参数详解

您可以通过以下参数评估预测结果的质量：

### 综合指标
通过综合指标，您可以快速判断预测任务整体表现，以此为基础决策是否要继续深入详细参数


| 指标 | 说明 |
|------|------|
| **综合排序得分 (Ranking Confidence)** | 结合多个指标计算的综合评分，得分越高预测越可信 |
| **原子冲突 (Atomic Clashes)** | 检测预测结构中是否存在不合理的原子空间重叠 |

### 结构层面指标

| 指标 | 衡量对象 | 分数范围 | 置信度等级 | 含义 |
|------|----------|----------|------------|------|
| **pTM** | 整个复合物结构 | > 0.75 | 高度可信 | 整体结构预测准确 |
| | | 0.5 - 0.75 | 大致可信 | 折叠方式基本正确 |
| | | < 0.5 | 不可靠 | 整体结构可能错误 |
| **ipTM** | 链间相对位置 | > 0.8 | 高质量 | 界面预测可靠 |
| | | 0.6 - 0.8 | 中等 | 界面预测基本正确 |
| | | < 0.6 | 可能失败 | 预测结果不可信 |
| **pLDDT** | 单个残基局部结构 | > 90 | 极高 | 深蓝色，局部结构非常可靠 |
| | | 70 - 90 | 高 | 浅蓝色，局部结构可靠 |
| | | 50 - 70 | 低 | 黄色，局部结构可信度低 |
| | | < 50 | 极低 | 红色，局部结构不可靠 |

### 置信度指标参考标准

| 指标名称 | 警告值阈点 | 含义说明 |
| ------------------ | ---------- | ------------------------------------------------------------ |
| Ranking confidence | < 0.5 | 综合置信度较低，结果可能不可靠 |
| ipTM | < 0.6 | 亚基相对位置预测可信度较低 |
| pLDDT | < 50 | 局部结构预测置信度极低 |
| pTM | < 0.5 | 整体结构预测与真实结构相似性较低 |

:::caution 注意
ipTM 对单链结构不具参考意义。
:::


### 链间关系分析解读

在“链间关系分析”部份，您可以通过以下信息判断预测表现

| 图示名称 | 功能说明 | 适用场景说明 |
| ------------------- | -------------------------------------------------- | ---------------------------- |
| PAE 图 | 残基对之间空间位置预测误差，颜色越深表示预测越可靠 | 链间/结构域间定位分析 |
| Chain ipTM 图 | 每条链与其他链的接口预测可信度 | 判断亚基装配是否合理 |
| Chain PAE（mean）图 | 计算链与其他链的平均空间误差 | 粗略评估链间定位质量 |
| Chain PAE（min）图 | 展示链间最小误差，识别潜在高可信度接口区域 | 接口分析、生物学功能推测 |
| Chain Clash 图 | 检测链间是否存在原子冲突（数值越大冲突越严重） | 排查结构冲突，确认结构合理性 |

:::tip 提示
对于单链结构，Chain ipTM、Chain PAE 和 Chain Clash 图不具参考价值。
:::



## 结果解读要点


1. **整体评估**：首先查看 Ranking Confidence 综合得分，得分 > 0.5 通常表示结果可信

2. **结构层面**：
   - pTM > 0.75 表示整体折叠可信
   - ipTM > 0.8 表示界面预测可靠（仅适用于复合物）

:::tip
3. **局部细节**：查看 pLDDT 分布，红色区域（< 50）的结构建议忽略或需要进一步验证
:::

4. **冲突检查**：确认 Atomic Clashes 为"无冲突"，有冲突的结构需要重新预测或验证输入



## 下载结果

### 文件格式

系统支持以下格式导出：

| 文件格式 | 说明 | 适用场景 |
|----------|------|----------|
| **PDB** | 生物大分子结构标准格式 | 通用结构分析工具 |
| **CIF** | PDBx/mmCIF 格式 | 更详细的结构信息 |

### 下载内容

| 内容 | 说明 | 文件类型 |
|------|------|----------|
| 预测的三维结构文件 | PDB 或 CIF 格式 | 结构文件 |
| 任务元数据信息 | 输入参数、模型版本等 | JSON |
| 置信度评估数据 | pLDDT、pTM、ipTM 等 | JSON/结构文件 |
