# 计费与价格


PaddleHelix 平台采用按模块计费模式。普通账户可免费升级为高级版，高级版支持提交计费任务。计费模块优先使用点券抵扣，余额不足时自动从百度云账户余额扣费。

本页面整合账号计费规则、计费模块、详细价格说明与欠费处理规则，便于快速查阅。


## 账号与计费规则

### 账号类型说明

| 账号类型 | 权限说明 |
|----------|----------|
| 普通账号 | 可免费升级为高级版 |
| 高级版账号 | 可提交计费任务，支持点券+余额混合支付 |
| 未接入计费模块 | 仅支持点券抵扣，不支持余额支付 |

更多说明请参考：账号等级与身份说明页面。


## 计费模块列表

目前支持计费的模块包括：

- 生物分子结构预测：HelixFold3  
- 虚拟筛选：HelixVS（仅支持商业分子库）  
- 分子设计：HelixVS-Syn  
- ADMET预测：HelixADEMT  
- RNA二级结构预测：LinearFold、LinearPartition  
- 序列设计：LinearDesign（mRNA）、HelixUTRdesign（5’UTR）  
- 小蛋白设计  
- 抗体设计  

*除上述模块外，其余模块暂不支持计费。*
