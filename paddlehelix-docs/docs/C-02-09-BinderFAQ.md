# 小蛋白设计 FAQ

在使用 HelixDesign-Binder 进行 Binder 设计的过程中，您可能会遇到各种关于设计模式选择、参数设置、结果评估等方面的问题。本页面汇集了用户最常询问的问题和答案，涵盖基础概念、输入配置、结果解读以及技术细节等各个方面，帮助您快速找到解决方案。如果本页未能解答您的疑问，欢迎通过官方渠道联系技术支持团队。



## 基础问题

### Q1: 什么情况下应该使用从头设计模式？

**从头设计**适用于：

- 没有已知的 binder 参考结构
- 需要探索全新的 binder 序列空间
- 对目标结合区域有明确偏好（可通过 hotspots 设置）

### Q2: 什么情况下应该使用基于参考模式？

**基于参考 Binder**适用于：

- 已有具有结合活性的 binder
- 有靶点-binder 复合物结构
- 需要保留 binder 的关键功能区域（motif）

### Q3: 可以设计多长的 Binder？

Binder 长度范围：**3-250 个氨基酸**

| 长度范围 | 适用场景 |
|----------|----------|
| 3-20 | 多肽/短肽 |
| 20-50 | 小型结合域 |
| 50-250 | 较大蛋白结合剂 |



## 结果问题

### Q4: 为什么没有产生满意的结果？

可能原因及解决方案：

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 结合分数低 | 靶点结构复杂 | 使用基于参考模式 |
| 界面预测不可靠 | 缺乏共进化信息 | 上传复合物结构 |
| 结果单一 | 参数设置过窄 | 放宽长度范围 |


### Q5: 如何判断结果是否可靠？

综合评估以下指标：

| 指标 | 推荐阈值 | 含义 |
|------|----------|------|
| Helix Binder Score | > 0.5 | 综合表现良好 |
| ipLDDT_Binder | > 70 | 界面预测可信 |
| ipTM | > 0.5 | 界面构象可信 |
| ΔG | < -5 kcal/mol | 结合稳定 |

**阈值参考表**：

| 分数区间 | 质量等级 | 说明 |
|----------|----------|------|
| Helix Binder Score > 0.5 | 良好 | 综合质量可接受 |
| ipLDDT_Binder > 70 | 可信 | 界面预测可靠 |
| ipTM > 0.5 | 合格 | 界面相对位置确定 |
| ΔG < -5 kcal/mol | 稳定 | 结合强度足够 |

## 技术问题


### Q6: 可以同时设计多个靶点吗？

不可以。每个任务只能针对一个靶点进行设计。如需多个靶点，请分别提交任务。



## 性能优化

### Q7: 如何提高设计成功率？

1. **选择合适的设计模式**
   - 有参考时使用基于参考模式
   - 无参考但有热点信息时使用从头设计

2. **合理设置参数**
   - 长度范围不要过窄
   - 根据应用需求调整最小长度

3. **充分利用参考结构**
   - 基于参考模式时上传复合物结构
   - 设置 motif 保留关键区域
