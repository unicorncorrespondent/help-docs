## 核心指标解读

### 核糖体载量均值 (MRL)

**MRL（Mean Ribosome Loading）** 是衡量翻译效率的核心指标：

| 指标 | 说明 |
|------|------|
| 定义 | 与给定 RNA 相关的核糖体平均数量 |
| 含义 | 翻译效率的代理指标 |
| 推荐 | MRL 越高，翻译效率越佳 |

**解读建议：**

1. **首要参考指标**：在选择设计结果时，MRL 应该是您最优先考虑的指标
2. **相对比较**：MRL 是相对指标，建议在多个设计结果中选择 MRL 最高的序列
3. **长度影响**：当 UTR 长度超过 100 nt 时，MRL 预测可能不够准确

**高质量 5'UTR 的特征：**
- 较高的 MRL 值
- 适中的 GC 含量（30-70%）
- 适度的 5'UTR-CDS 配对（避免过度配对抑制翻译）

### 配对数量统计

5'UTR 配对数量反映了 5'UTR 与 CDS 区域之间的碱基配对情况：

| 配对类型 | 含义 |
|----------|------|
| 正常配对 | 标准 Watson-Crick 配对（A-U, G-C） |
| 非标准配对 | 非经典的碱基配对（如 G-U wobble） |

**解读提示：**
- 过多的 5'UTR-CDS 配对可能形成阻碍翻译的结构
- 适度的配对可以稳定 mRNA 二级结构
- 建议选择配对数量适中的结果

### 相似性

当设置了参考 5'UTR 时，结果会显示设计结果与参考序列的相似程度：

| 相似性范围 | 解读 |
|------------|------|
| >80% | 与参考序列非常接近 |
| 50-80% | 在保留关键特征的同时有所变化 |
| <50% | 差异较大，引入了新的设计特征 |

**使用场景：**
- 如果需要保留某些已知功能元件，选择高相似性的结果
- 如果需要创新设计，可以考虑较低相似性的结果

### GC 含量

GC 含量影响 mRNA 的稳定性和翻译效率：

| GC 含量 | 特点 |
|---------|------|
| 过低 (<30%) | 序列可能不稳定，容易降解 |
| 适中 (30-70%) | 较为理想，兼顾稳定性和灵活性 |
| 过高 (>70%) | 可能形成过于稳定的结构，影响翻译 |

**建议：**
- 优先选择 GC 含量在 30-70% 范围内的结果
- 避免选择 GC 含量过低（<25%）的结果

## 二级结构可视化

### 交互式结构图

点击结果前方的单选框，可以在表格下方查看该 5'UTR 及其 CDS 形成的二级结构图。这是评估设计结果的重要辅助工具，可以帮助您直观地了解 5'UTR 的结构特征。

![Secondary Structure](https://bml-test-test.bj.bcebos.com/helix_upload/lineardesign%20%E7%BB%93%E6%9E%9C%E9%A1%B5%20advance%20%E5%B7%A6%E8%BE%B9%E6%A0%8F%20%E4%B8%AD%E6%96%87-9bf23cc6-6eca987c.jpeg)

### 交互功能

- **鼠标悬停**：查看特定碱基或配对的详细信息
- **序列高亮**：与参考序列不同的核苷酸会被高亮显示

### 解读提示

1. **避免强二级结构**：5' 末端不应该形成过于稳定的二级结构，这会影响核糖体结合
2. **关注 CDS 起始区域**：AUG 附近的结构对翻译起始特别重要
3. **比较不同结果**：使用结构可视化对比不同设计结果的差异
