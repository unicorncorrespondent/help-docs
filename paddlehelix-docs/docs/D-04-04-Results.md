# 查看结果

> 解读分子设计结果与各项指标

完成分子设计任务后，您可以在结果页面查看生成的分子的各项指标和性质。

## 结果页面概览

结果页面顶部会显示本次设计的输入信息：

- 靶蛋白信息
- 骨架结构
- 生长位点设置

您可以：

- 切换卡片视图或 3D 结构视图
- 更改结果的排序方式
- 筛选感兴趣的结果

## 卡片视图

卡片视图以卡片形式展示每个生成分子的关键属性。

### 分子属性

每个卡片显示四个核心指标：

| 指标 | 含义 | 理想范围 |
|------|------|----------|
| **活性打分** | 分子与蛋白的亲和力预测 | 越高越好 |
| **可成药性 (QED)** | 化合物成为候选药物的潜质 | 0-1，越高越好 |
| **易合成度 (SA)** | 化合物的合成难度评估 | 1-10，越低越好 |
| **ClogP** | 脂水分配系数 | 适中为宜 |

**指标说明：**

- **活性打分 (Active Score)**：预测药物与靶蛋白之间的亲和力分数
- **可成药性 (QED)**：定量评估药物相似性，0-1，实数越高越好
- **易合成度 (SA)**：基于片段复杂度的评估，1-10，数字越大越难合成
- **ClogP**：正辛醇/水分配系数的对数，影响药物的吸收和分布

![Card View](https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA7g2e3JYWBLJZFfGb7nmDK8.png)



点击卡片可查看更详细的信息：

- **SMILES 分子式**：分子的化学结构描述
- **分子量 (MW)**：分子质量（g/mol）
- **天然产物相似度 (NP)**：评估分子与天然产物的结构相似性，-5 到 5，越高越可能是天然产物

![Detail View](https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA50DOM4hEVFbZWGp0RWE9ez.png)

## 3D 结构视图

3D 结构视图展示分子的三维构象与靶蛋白的相互作用。

### 查看构象

- 点击左侧分子图或展开子层级点击任一 pose
- 3D 视窗中显示对应的分子构象
- 分子后面默认展示活性打分 A

![3D View](https://bml-test-test.bj.bcebos.com/helix_upload/%E5%88%86%E5%AD%90%E7%94%9F%E6%88%90%20%E7%BB%93%E6%9E%9C%E9%A1%B5%203d%20%E4%B8%AD-57003b63.jpg)

### 切换展示指标

点击表头的下拉按钮，可以切换展示不同的结果指标。

### 多结构对比

如需同时查看多个结构：

1. 悬浮并点击分子前的眼睛图标
2. 可同时在画布上查看多个结构
