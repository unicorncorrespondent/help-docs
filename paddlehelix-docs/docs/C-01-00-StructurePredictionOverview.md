# 结构预测

> 了解如何使用 HelixFold3 和 HelixFold-S1 预测各种生物分子的三维结构与相互作用

蛋白质是生命活动的执行者，而它们的功能取决于精确的三维空间结构。从氨基酸序列预测蛋白质结构是现代生物学的核心挑战之一，也是药物研发的关键环节。随着人工智能技术的迅猛发展，以 AlphaFold 系列模型为代表的生物大模型已经显著地推动了该领域的进展。在本页，我们将简要介绍生物结构预测的相关背景，并展示 HelixFold3 与 HelixFold-S1 的核心能力与技术突破。


## 生物分子结构预测  

生物分子的三维结构决定其功能与相互作用，是理解生命活动、疾病机制和药物设计的核心基础。然而，受实验条件限制，许多蛋白质和核酸结构难以高效解析，因此需要借助计算与人工智能方法进行结构预测，以提升获取效率并弥补实验不足。

### 传统方法的局限  

蛋白质结构解析主要依赖 **X射线晶体衍射、核磁共振（NMR）和冷冻电镜（Cryo-EM）** 等技术。虽然精度高，但流程复杂、成本昂贵，往往需要高纯度样品和长时间实验周期，且部分蛋白质难以成功测定结构，限制了结构数据的获取规模。

### AI 带来的变革  

深度学习的发展，特别是 AlphaFold 等模型的出现，改变了结构研究方式。AI 方法可在短时间内完成高精度结构预测，显著降低成本，并支持大规模结构解析，推动生命科学向数据驱动与智能预测方向发展。

## HelixFold3

在人工智能推动结构预测快速发展的背景下，各类高性能模型不断涌现，进一步拓展了生物分子结构预测的应用边界。其中，**HelixFold3** 是百度螺旋桨团队推出的新一代生物分子结构预测模型，面向更加复杂的多分子相互作用场景进行设计与优化。该模型在多种生物分子体系中均表现出优异的预测能力，包括蛋白–蛋白、蛋白–小分子、蛋白–RNA 以及蛋白–DNA 等复合体系，展现出良好的泛化性能与结构解析精度，为多类型分子互作研究与药物研发提供了有力支持。

与 AlphaFold3 相比，HelixFold3 支持**更复杂的生物分子相互作用场景**，包括共价结合、化学修饰、PROTAC（蛋白降解靶向嵌合体）、分子胶和金属酶等。不仅满足了学术研究中对更**高精度和多样化**场景的需求，也显著提升了产业界在**药物研发中的效率和精准度**。HelixFold3 能够帮助用户**准确、高效**地设计和模拟复杂分子，**加速**技术成果的转化，**迅速缩短创新产品的上市时间**。

### HelixFold3 支持分子类型

| 分子类型 | 示例 |
|----------|------|
| 蛋白质 | 单链蛋白、蛋白复合物 |
| DNA | 单链/双链脱氧核糖核酸 |
| RNA | 核糖核酸分子 |
| 小分子配体 | 药物分子、辅因子 |
| 离子 | 金属离子等 |

### HelixFold3 特色与优势

- **高精度预测**：在多种分子体系上达到国际领先水平
- **广泛的配体支持**：兼容CCD/SMILES格式，支持PROTAC等新药开发，提升早期药物发现效率
- **灵活的序列修饰**：支持蛋白质/DNA/RNA自定义修饰（SMILES），覆盖PDB中50%修饰场景
- **相互作用约束**：可添加实验/理论约束条件，降低研发成本 
- **参考结构增强**：整合已知晶体结构，提升预测可靠性  
- **自定义参数**：可调Ensemble数量/Recycle次数，平衡精度与效率，优化难预测区域结果 
- **API与SDK**：为企业级客户提供 API/SDK 接口，支持按量调用、免 GPU 硬件投入，兼容现有工具链

## HelixFold-S1 介绍

尽管已有显著进展，在某些复杂场景中，例如缺乏丰富共进化信息（如抗原–抗体结构预测），或结合模式不在训练集中（如 De novo 分子设计），当前主流模型（如 AlphaFold3、HelixFold3、Chai-1）仍面临预测精度不足的问题。因此，在HelixFold3 基础上，我们开发了增强版本HelixFold-S1。

###  HelixFold-S1 核心创新：接触感知采样

S1 模型引入了一种类似大语言模型"推理时计算"（Test-time Scaling）的策略：

1. **评估接触概率**：在开始预测前，先评估每对残基发生物理接触的可能性
2. **高概率区域优先**：模型会优先围绕高接触概率的区域进行建模
3. **提高界面精度**：使多链蛋白质的界面预测更加准确

###  HelixFold-S1 适用场景

- **抗原-抗体复合物**：预测抗体如何结合靶点
- **蛋白-蛋白相互作用**：研究信号通路中的蛋白相互作用
- **多亚基复合物**：分析大型蛋白机器的结构
