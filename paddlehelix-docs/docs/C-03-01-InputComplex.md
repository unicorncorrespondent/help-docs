# 输入复合体结构

> 上传抗原-抗体复合物结构文件

完成抗体设计任务的第一步是输入待设计的抗原-抗体复合物结构。本节将详细介绍如何通过 PDB ID 搜索或直接上传 .cif 格式文件来完成结构输入，以及系统对结构完整性的检查要求和自动修复机制。正确上传符合要求的结构文件是获得高质量设计结果的前提条件。


## 数据准备要求

在使用 HelixDesignAntibody 进行抗体设计前，您需要准备一个包含至少一条抗原链（目标靶点）和一条抗体重链的复合体结构。为确保设计效果，请尽量满足以下条件：

:::tip
| 完整性要求 | 说明 |
|------------|------|
| **抗原链** | 应尽可能完整。若有缺失，系统尝试自动补全，失败可能影响效果 |
| **CDR 结构** | 必须完整，否则无法进行设计 |
| **Fv 区域** | 应尽可能完整。缺失 ≤2 个氨基酸可尝试修复；>2 个无法设计 |
| **Fc 区域** | 建议完整。缺失 >2 个无法进行全链设计，Fc 区域将从结果中移除 |
:::



## 上传方式
在了解数据准备要求后，您可以选择以下两种方式进行结构上传。

### 方式一：搜索 PDB 数据库

1. 在 PDB ID 输入框中输入 4 位编号（如 `7QUH`）
2. 从下拉列表中选择目标结构
3. 系统自动加载并预览结构

### 方式二：上传 .cif 文件

1. 点击"文件上传"按钮
2. 选择 `.cif` 格式文件
3. 系统自动加载并预览结构

<img src="https://bml-test-test.bj.bcebos.com/helix_upload/%E6%8A%97%E4%BD%93%E8%AE%BE%E8%AE%A1%20%E8%BE%93%E5%85%A5%E7%BB%93%E6%9E%84%20%E4%B8%8A%E4%BC%A0%E6%96%87%E6%A1%A3%20%E4%B8%AD%E6%96%87-b96ee9c9.jpg" style="zoom:80%;" />

上传完成后，系统将对结构进行完整性检查，以确保满足设计要求。

## 结构完整性检查

上传后系统会自动检查：

| 检查项 | 处理方式 |
|--------|----------|
| 抗原链缺失 | 自动尝试补全 |
| CDR 区域缺失 | 无法进行设计 |
| Fv 区域缺失 ≤2 aa | 尝试自动修复 |
| Fv 区域缺失 >2 aa | 拒绝任务 |
| Fc 区域缺失 >2 aa | 移除 Fc 区域后设计 |
