# 常见问题

**解答 HelixFold3/S1 使用中的常见疑问**

在使用 HelixFold 进行结构预测的过程中，您可能会遇到各种问题。本页面汇集了用户最常询问的问题和答案，涵盖模型选择、功能使用、数据格式等各个方面。无论您是初次使用的新用户还是有一定经验的研究人员，都可以在这里找到有帮助的信息。如果本页未能解答您的疑问，欢迎通过官方渠道联系我们的技术支持团队。


**Q: HelixFold3 和 HelixFold-S1 有什么区别？**

A: HelixFold3 是通用模型，适用于大多数场景；HelixFold-S1 专为多链复合物优化，尤其在抗原-抗体场景表现更佳。

**Q: 如何选择模型？**

A: 一般预测选择 HelixFold3；抗原-抗体或多链蛋白界面预测选择 HelixFold-S1。

**Q: 支持哪些修饰类型？**

A: HelixFold3 支持蛋白质、DNA、RNA 的自定义侧链修饰；HelixFold-S1 暂不支持修饰。

**Q: 结果文件包含哪些内容？**

A: 包含 PDB/CIF 格式的结构文件、置信度评分等元数据。
