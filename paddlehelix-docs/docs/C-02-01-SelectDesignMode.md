# 选择设计模式

:::info 分类
输入
:::


**从头设计 vs 基于参考 Binder 设计**

HelixDesign-Binder 提供两种不同的 Binder 设计模式，分别适用于不同的研究场景。**从头设计模式**适合在没有已知 Binder 参考结构的情况下，从零开始生成全新的 Binder 分子。而**基于参考 Binder 模式**则适合已有已知 Binder 或复合物结构，需要在现有基础上进行优化的情况。本页面将详细介绍这两种设计模式的特点和适用场景，帮助您根据研究需求选择最合适的设计策略。

## 两种设计模式对比

| 特性 | 从头设计 | 基于参考 Binder |
|------|----------|----------------|
| 输入要求 | 只需靶点结构 | 需要靶点 + 参考 binder 结构 |
| 适用场景 | 无已知 binder | 有已知 binder 或复合物结构 |
| 热点设置 | 支持 | 不支持 |
| Motif 固定 | 不支持 | 支持 |

## 从头设计模式

基于您提供的靶点结构，**从零生成全新的小蛋白 binder**。

### 特点

- 无需提供任何 binder 参考结构
- 可设置**热点氨基酸 (hotspots)**，引导 binder 优先结合靶点特定区域
- 适合探索全新的 binder 设计空间

### 热点氨基酸设置

如果您希望引导模型重点关注靶点上的特定结合区域：

1. 打开热点设置开关
2. 点击序列中的氨基酸即可将其设为 hotspot
3. **最多可设置 8 个 hotspot 位点**

热点区域会被高亮显示，binder 设计时会优先覆盖这些区域。

## 基于参考 Binder 模式

在已有 binder 的基础上，对其进行**结构或序列优化**。

### 适用场景

- 已知有结合活性的 binder，需要提高亲和力或稳定性
- 有靶点-binder 复合物结构，需要优化界面
- 需要保留 binder 的关键功能区域（motif）

### Motif 设置

如果您希望在设计过程中**固定 binder 中的关键结构或功能片段**：

1. 打开 motif 开关
2. 点击序列中的起始和结束氨基酸，选中连续区域作为 motif
3. **最多可设置 5 个 motif**

<img src="https://bml-test-test.bj.bcebos.com/helix_upload/%E5%B0%8F%E8%9B%8B%E7%99%BD%E8%AE%BE%E8%AE%A1-%E8%BE%93%E5%85%A5-%E8%AE%BE%E7%BD%AEmotif-%E4%B8%AD%E6%96%87-2bf4dff1.jpg" style="zoom:100%;" />

### Motif 约束类型

| 约束类型 | 说明 |
|----------|------|
| 保证结构不变 | 保持该区域的三维结构 |
| 保证序列不变 | 保持该区域的氨基酸序列 |
| 结构和序列都不变 | 同时保持结构和序列 |

## 选择建议

- **对目标 binder 结构一无所知** → 选择**从头设计**
- **有明确的结合活性参考 binder** → 选择**基于参考 Binder**
- **需要保留 binder 的关键功能区域** → 使用 motif 设置
