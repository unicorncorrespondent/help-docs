# 分子库配置

> 选择或上传虚拟筛选的化合物库

准备好蛋白质结构和结合位点后，下一步需要选择用于筛选的化合物库。分子库的选择直接影响虚拟筛选的覆盖范围和结果质量。商业分子库提供了丰富的化合物资源，而自定义分子库则允许您针对特定研究方向进行精准筛选.

## 分子库来源

HelixVS 集成了多个商业化的预置分子库，同时也支持用户上传自定义的分子集合。

### 预设商业分子库

HelixVS 集成了多个商业分子库，这些库经过质量控制，化合物纯度和多样性都有保障：

1. 在「选择分子库」区域勾选您需要使用的分子库名称

2. 设置「最大输出分子数量」，即您希望从该库中筛选的最大分子数

> ⚠️ 注意：实际筛选结果可能少于您设定的数量，因为部分分子可能因为结构不兼容或不符合筛选条件而被过滤掉


以下为 HelixVS 系统预设的分子库

| 分子库名称 | 分子数量 | 价格 |
|------------|----------|------|
| Targetmol | 3.4 万 | 70 元/次 |
| Lifechemicals | 82 万 | 1200 元/次 |
| ChemDiv | 283 万 | 3700 元/次 |
| topscience | 1687 万 | 19000 元/次 |


### 自定义分子库

如果预设商业分子库不能满足您的研究需求，或者您需要针对特定靶点筛选特定的化合物集合，可以上传自定义分子库。

#### 方式一：文本框输入

对于少量分子（最多 1000 个），可以直接在文本框中输入 SMILES：

- 每个 SMILES 占据一行
- 不同分子之间使用换行符分隔

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA5UAEcNucBNx64hjYURYj-d.png" style="zoom:50%;" />

**绘制分子辅助**

如果您不确定某个分子的 SMILES 写法，可以点击「绘制分子」打开分子绘制面板：

1. 使用绘图工具画出目标化合物的结构
2. 点击「确认」按钮
3. 系统会自动将绘制的结构转换为 SMILES 并填入文本框

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA4YX69QIRhEraOH4-6PoDfj.png" style="zoom:33%;" />

#### 方式二：上传文件

对于大量分子或需要反复使用的分子集合，建议上传文件，我们支持`.csv`、`.txt` 与.`smi`格式 ，且需要满足以下要求：

| 要求 | 说明 |
|------|------|
| 行数 | 每行一个 SMILES |
| 数量限制 | 最多 1000 个 SMILES |
| 长度限制 | 每个分子不超过 200 个字符 |
| 文件大小 | 不超过 10 MB |

您可以点击「上传范本下载」获取示例文件格式

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA4XETKaHjtJwZ1vdrXV4Yx1.png" style="zoom:50%;" />
