# 筛选参数设置

> 配置虚拟筛选的条件与约束



通过指定配体与蛋白之间必需的相互作用，可以筛选出符合特定药效团要求的分子。HelixVS 提供灵活的参数设置功能，让您能够精确控制筛选过程，确保筛选结果符合您的研究需求。

以下为 HelixVS 支持的相互作用类型：

| 缩写 | 相互作用 | 说明 |
|------|----------|------|
| **Hbond** | 氢键 | 供体-受体之间的静电吸引 |
| **Hydrophobic** | 疏水作用 | 疏水基团之间的相互作用 |
| **PiPi** | π-π堆积 | 芳香环之间的相互作用 |
| **PiCation** | π-阳离子作用 | 芳香环与带电基团的相互作用 |


结合模式的书写需要满足以下格式：

```
模式_氨基酸_位置

例如：
- Hbond_CYS_108     → 与108号半胱氨酸形成氢键
- Hydrophobic_ALA_107 → 与107号丙氨酸形成疏水作用
- PiPi_ALA_107       → 与107号丙氨酸形成π-π堆积
- PiCation_ALA_107   → 与107号丙氨酸形成π-阳离子作用
```

同时可以加入逻辑运算符增加多种相互作用

| 运算符 | 含义 | 说明 |
|--------|------|------|
| `&` | 与 | 两个条件必须同时满足 |
| `|` | 或 | 满足任一条件即可 |
| `!` | 非 | 排除满足条件的分子 |

### 约束示例

| 条件表达式 | 含义 |
|------------|------|
| `Hbond_CYS_108 \| Hydrophobic_ALA_107` | 与 CYS_108 形成氢键 **或** 与 ALA_107 形成疏水作用 |
| `Hbond_CYS_108 & Hydrophobic_ALA_107` | 与 CYS_108 形成氢键 **且** 与 ALA_107 形成疏水作用 |
| `!Hbond_CYS_108` | **不** 与 CYS_108 形成氢键 |
| `(Hbond_CYS_108\|Hydrophobic_ALA_107)&PiPi_ALA_107` | 满足前两个条件之一 **且** 与 ALA_107 形成 π-π 堆积 |
