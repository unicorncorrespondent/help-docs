> 通过"一键快速体验"功能，快速体验螺旋桨 Paddle Helix 生物计算平台

螺旋桨平台提供了"一键快速体验"功能，让您无需了解复杂的参数设置，即可立即体验平台的强大生物计算能力。我们提供多个示例数据，帮助您快速了解平台具体操作流程。无论您想使用蛋白结构预测、mRNA设计还是小分子筛选，都可以通过这一入口快速上手。


![Example tutorial](/images/B-00-QuickStart.png)

## 体验流程
- 点击螺旋桨平台任意页面（示例：[小蛋白设计](https://paddlehelix.baidu.com/app/all/helixdesign-binder/forecast)）
- 在页面右上角“一键快速体验”中，任意点击一个示例
- 点击“下一步”或“提交运算”
- 在“任务管理”中查看结果


## 下一步
