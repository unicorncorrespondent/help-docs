# 上传参考结构

> 利用已知结构提升预测精度

参考结构功能可以帮助您提升蛋白-蛋白、小分子、核酸复合物的预测精度，提升模型表现，当您的实体属于以下情况时，上传参考结构对模型预测精度的提高会更加显著：

- **蛋白-蛋白/小分子/核酸复合物**：已知蛋白部分晶体结构，如靶点结构
- **蛋白突变体**：需基于野生型结构预测，且蛋白与野生型配体结构已知

您可以在以下场景中考虑上传参考结构：
- 已知蛋白部分晶体结构（如靶点结构）
- 蛋白突变体预测（基于野生型结构）
- 提高特定复合物的预测精度




## 使用步骤

HelixFold3 支持上传参考结构功能，允许您输入 PDB ID 直接从蛋白质数据库获取结构，或者上传本地的 .cif 格式结构文件。您可以在一次预测任务中上传多个参考结构，系统会在 3D 视图中高亮显示参考链，方便您确认并将其关联到对应的预测实体上。


1. 在完成实体序列输入后，开启"上传参考结构"开关

<img src="https://bml-test-test.bj.bcebos.com/helix_upload/image-2-a13dc437.png">

2. 输入4位**PDB ID**（如`8FAX`），或本地 .cif 文件上传，参考链需≥6个残基，且与预测实体序列相似度≥90%

<img src="https://bml-test-test.bj.bcebos.com/helix_upload/image-2-fa151334.png" alt="3"  style="width:700px" />

3. 在右侧链列表中，勾选参考链（在3D视图中高亮显示），具体选择技巧见下方“参考链选择技巧”

<img src="https://bml-test-test.bj.bcebos.com/helix_upload/image-2-3e0d63ad.png">

4. 在"对应实体"列下拉菜单，选择当前预测的蛋白实体，关联对应的预测实体

> ⚠️ 注意：目前只支持为蛋白链指定参考结构，因此下拉菜单中只会包含类型为“蛋白”的实体



### 参考链选择技巧

- 需要模型同时参考链内构象+链间结构 → 同一参考结构选多条链
- 仅需模型参考链内构象 → 分次上传同一参考结构选单链

![9](https://bml-test-test.bj.bcebos.com/helix_upload/%E6%B5%81%E7%A8%8B%E5%9B%BE-202503261746-1acbd640.png)

## 其他操作

如果您需要删除已上传的参考结构，可以点击待删除行后方的三个点，然后点击“删除（删除该参考链）”或“清空（清空整个参考结构）”。如果您需要上传多个参考结构，可以点击“增加一组”。
