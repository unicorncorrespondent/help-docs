# 选择设计链和区域

> 配置抗体的设计参数

上传结构后，您需要配置抗体设计的具体参数，包括选择待设计的抗体链和指定设计区域模式。本节将介绍从选择目标设计链到配置设计区域的全流程，帮助您根据实验需求选择全链设计、完整 Fv 区域设计或指定 Fv 区域设计等不同模式。


## 选择目标设计链


1. 从下拉菜单中选择待设计的抗体链
2. 系统默认选择**第一条抗体重链**
3. 确认所选链的 CDR 区域完整，否则无法进行设计


## 选择设计区域

完成目标设计链选择后，接下来需要确定具体的设计区域，您可以从以下三个区域进行选择并设计

| 设计区域 | 说明 | 适用场景 |
|----------|------|----------|
| **全链设计** | 对整个抗体链进行序列设计 | 全面优化抗体性能 |
| **完整 Fv 区域** | 仅设计 Fv 可变区 | 保留 Fc 恒定区 |
| **指定 Fv 区域** | 仅设计 CDR 等特定区域 | 定点突变、亲和力成熟 |

### 指定 Fv 区域格式

在输入框中输入残基索引或范围，多个项用逗号分隔：

```
示例：80-90,100,120-130
```

> **注意**：数字应对应 `.cif` 文件中的残基编号

在了解设计区域的选择方式后，请注意以下参数限制以确保任务能够正常运行。

## 参数限制

| 参数 | 限制 | 说明 |
|------|------|------|
| **总链长** | ≤ 1000 | 抗原链 + 设计链 + 配对链总和 |
| **Fv 缺失** | ≤ 2 aa | 可尝试修复；>2 aa 无法设计 |
| **Fc 缺失** | > 2 aa | 无法进行全链设计 |
