# 定义结合位点

> 设定蛋白口袋进行分子对接

确定蛋白质上的配体结合位置是虚拟筛选的关键步骤之一。结合位点（也称"口袋"或"结合口袋"）是蛋白表面适合小分子配体结合的凹陷区域，只有准确定义这个区域，分子对接程序才能评估小分子与蛋白的结合模式。

## 定义方式

HelixVS 提供两种定义结合位点的方式：使用 AI 模型自动预测可能的结合口袋，或手动输入坐标和尺寸精确定义。本节将详细介绍这两种方法的具体操作和适用场景。

如果您输入了不止一个蛋白信息，则可以以此按照同样方法进行结合位点定义 继续为其定义结合位点，方法相同。

### 方式一：预测位点

HelixVS 使用深度学习模型预测蛋白质表面可能的结合口袋，这是一种快速便捷的位点定义方式，特别适合缺乏先验信息的探索性筛选。


1. 点击「预测位点」标签切换到预测模式

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA7fAqHmwspMkIxu038d3EHo.png" style="zoom:80%;" />

2. AI 模型会分析蛋白质结构并返回预测结果列表，每个预测结果包含以下信息：

| 列 | 说明 |
|----|------|
| 预测名称 | 位点的编号标识 |
| 口袋打分 | 结合可能性评分（越高越可能是真实口袋） |
| 操作 | 可下载该口袋的配置文件 |

3. **选择口袋**：点击预测名称列表中的某一行，该口袋会以黑色立方体的形式显示在 3D 视窗中，帮助您直观查看口袋的位置和范围

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA6H_m3B-bdIuawSZbogzQzQ.png" style="zoom:50%;" />



### 方式二：手动输入

如果您已经知道口袋的精确位置（例如从文献或共结晶结构中获知），可以手动输入参数精确定义结合口袋。



1. 在第二步页面中，点击「手动输入」标签切换到手动输入模式

2. 输入口袋中心坐标（X, Y, Z）和长宽高尺寸（W, H, L）

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA5jukr_QD9AQajgGEQv_-tb.png" style="zoom:50%;" />


3. 点击「显示口袋」按钮，系统会在 3D 视窗中显示定义的口袋区域

4. 检查口袋是否正确覆盖了预期的结合区域，如有必要调整参数重新显示

<img src="https://easydl-download.bj.bcebos.com/helix_doc/AgAACoBIFA7ALVIkLTlNFLf3roGpfhw5.png" style="zoom:50%;" />

#### 必要参数

| 参数 | 说明 | 要求范围 |
|------|------|----------|
| X, Y, Z | 口袋中心的空间坐标 | 必须在蛋白坐标范围内 |
| W (宽) | X 轴方向尺寸 | 10-30 Å |
| H (高) | Y 轴方向尺寸 | 10-30 Å |
| L (长) | Z 轴方向尺寸 | 10-30 Å |

#### 注意事项

- 每次修改坐标或尺寸后，必须重新点击「显示口袋」才能更新 3D 视窗中的显示
- 口袋范围不能超出蛋白结构，否则可能导致对接计算失败
- 如果有共结晶配体的参考结构，建议以配体中心作为口袋中心

## 使用示例位点

如果您在第一步使用了示例蛋白，平台会预设该蛋白的典型结合位点。您可以直接点击「使用示例」调用预设的位点配置，无需重新预测或手动输入。
