# mRNA 药物研发

> 用 AI 技术进行 mRNA 序列设计、优化和结构预测

mRNA 药物研发是当前生物医药领域最具潜力的新兴技术之一。从 mRNA 疫苗到蛋白替代疗法，mRNA 技术正在革新疾病预防和治疗方式。PaddleHelix 提供完整的 mRNA 药物研发工具链，帮助研究人员利用人工智能技术设计和优化 mRNA 序列，提升 mRNA 的稳定性和翻译效率，从而加速药物研发进程。


## mRNA 药物研发工具箱

PaddleHelix 的 mRNA 药物研发工具涵盖了从序列设计到结构验证的完整工作流程。无论您是需要设计编码特定蛋白的 mRNA 序列，还是需要优化 5'UTR 区域以提升翻译效率，或者需要预测 RNA 二级结构以确保序列稳定性，都可以在我们的平台上找到合适的工具。


| 工具 | 功能 | 应用场景 |
|------|------|----------|
| **[LinearDesign](./E-05-01-LinearDesignOverview)** | mRNA 序列设计 | 优化稳定性(MFE)和翻译效率(CAI) |
| **[HelixUTRDesign](./E-04-01-HelixUTRDesignOverview)** | 5'UTR 序列生成 | 提升翻译起始效率 |
| **[LinearFold](./E-02-01-LinearFoldOverview)** | RNA 二级结构预测 | 快速预测 RNA 折叠结构 |
| **[LinearPartition](./E-03-01-LinearPartitionOverview)** | 碱基配对概率预测 | 分析结构分布和可能性 |
