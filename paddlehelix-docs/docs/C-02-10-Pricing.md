# 小蛋白设计计费与价格

<div class="doc-page-type doc-page-type--other"><span class="doc-page-type__icon">🧭</span><span>分类：其他</span></div>


小蛋白设计采用**算力单元**计费模式，按照实际消耗的算力进行收费。

## 算力单元说明

算力单元是 PaddleHelix 平台用于衡量计算资源消耗的单位。不同任务类型、输入数据量、模型参数等都会影响算力单元的消耗量。

具体算力单元的计算规则和定义，请参阅[计费与价格](F-02-02-Price)页面。


## 支付方式

小蛋白设计支持以下支付方式：

| 支付方式 | 说明 |
|----------|------|
| 点券 | 优先抵扣 |
| 百度云账户余额 | 余额支付（高级版支持） |
