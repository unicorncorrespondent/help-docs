# 查看结果与参数解读

:::tip 分类
结果
:::


**解读抗体设计结果的评分指标**

抗体设计完成后，系统会输出大量候选序列及其预测结果。本节将详细介绍如何解读设计结果表格中的各项评分指标，包括预测结合能 ΔG、pLDDT 置信度、ipTM 界面得分等核心参数的含义和推荐筛选标准。

#

## 在线页面结果展示

### 结果展示

提交后系统以表格形式展示设计结果：

### 表格字段说明

| 指标 | 全称 | 范围 | 说明 | 
|------|------|------|------|
| **ID** | - | 0-99 | 设计结果编号 | 
| **类型** | Type | Wild type / Designed | 野生型或设计型序列 | 
| **抗体序列** | - | - | 设计的抗体氨基酸序列 | 
| **预测结合能** | ΔG | 越负越好 | kcal·mol⁻¹，亲和力指标 | 
| **PAE** | - | ≥0，越小越好 | Å，残基位置预测误差 | 
| **pLDDT** | - | 0-100，>90 高置信度 | 局部结构预测置信度 | 
| **Fitness** | - | -∞~0，越大越好 | 序列进化适应性 | 
| **ipTM** | - | 0-1，>0.8 高置信度 | 界面构象可靠性 | 
| **Ranking_confidence** | - | 0-1，越高越好 | 综合可信度评分 | 

### 深入分析

点击"查看"可查看详细信息：

- **PAE 图**：残基间相对位置预测误差热力图
- **三维结构图**：pLDDT 着色的构象可视化
- **相互作用环形图**：界面残基相互作用类型统计
- **生化指标**：表面残基性质分析

### pLDDT 颜色编码

三维结构图中使用 pLDDT 值着色：

| 颜色 | pLDDT 区间 | 置信度等级 | 含义 |
|------|------------|------------|------|
| 深蓝 | >90 | 极高 | 非常高置信度 |
| 浅蓝 | 70-90 | 高 | 高置信度 |
| 黄色 | 50-70 | 低 | 低置信度 |
| 橙红 | <50 | 极低 | 非常低 |

## 核心评分指标 

### 指标参考表

| 指标类别 | 指标 | 含义 | 参考标准 |
|----------|------|------|----------|
| **结合亲和力** | 预测结合能 ΔG | 预测的结合自由能 | < -5 kcal/mol 为佳 |
| | Fitness | 序列进化适应性 | 越大越好 |
| **结构可靠性** | pLDDT | 单个残基局部置信度 | >90 高，70-90 中，<50 低 |
| | ipTM | 界面相对位置可信度 | >0.8 高置信度 |
| | PAE | 预测对齐误差 | 越小越好 |

### 相互作用类型

环形图展示 6 种残基相互作用类型：

| 相互作用类型 | 说明 |
|-------------|------|
| 带电/带电 | 静电相互作用 |
| 带电/极性 | 静电+氢键 |
| 带电/非极性 | 静电+疏水作用 |
| 极性/极性 | 氢键网络 |
| 极性/非极性 | 混合作用 |
| 非极性/非极性 | 疏水作用 |

相互作用数量越多、类型越丰富，结合强度通常越强。

### 生化指标

| 指标 | 说明 |
|------|------|
| 非相互作用表面残基占比 | 影响整体亲和力 |
| 非极性表面残基占比 | 与结合强度正相关 |
