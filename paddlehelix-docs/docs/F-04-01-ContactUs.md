## 平台内在线反馈

如在使用过程中遇到问题，建议优先通过平台内反馈提交：

1. 登录 PaddleHelix 平台  
2. 点击页面右下角 **「反馈」** 按钮  
3. 填写问题并提交  

参与用户调研有机会获得平台免费使用额度。

## 问题反馈建议内容

为便于快速定位问题，您可以在反馈时提供以下信息：

| 信息 | 说明 |
|------|------|
| 问题描述 | 具体问题或报错情况 |
| 操作步骤 | 可复现问题的完整流程 |
| 截图或日志 | 错误截图或日志信息 |
| 环境信息 | 浏览器、系统版本等 |

信息越完整，处理效率越高。
