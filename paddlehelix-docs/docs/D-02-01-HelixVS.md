# HelixVS 虚拟筛选

> 基于深度学习的千万/亿级小分子虚拟筛选

虚拟筛选（Virtual Screening）是药物研发早期阶段的重要技术，通过计算方法评估大量化合物与靶点的结合潜力，从而筛选出最有希望的候选分子进行后续实验验证。与传统的实验筛选相比，虚拟筛选具有成本低、速度快、通量高的显著优势，能够在药物发现初期排除大部分活性较低的分子，使研究人员可以将资源集中在最有潜力的候选化合物上，从而显著提升研发效率与成功率。

HelixVS 是百度螺旋桨团队研发的一套完整端到端虚拟筛选系统，能够实现千万至亿级别的小分子虚拟筛选。该系统旨在提高药物研发过程中活性分子的筛选效率与成功率，并显著缩短药物研发周期、降低研发成本。通过构建完整的虚拟筛选流程，HelixVS 可以从超大规模虚拟化合物库中快速筛选出针对目标靶点具有潜在高活性的候选分子，为药物发现提供强大的计算支持。


## HelixVS 核心优势

HelixVS 的核心优势在于其在大规模筛选场景下的卓越性能表现。系统支持千万至亿级分子库的高通量筛选，在保持高精度的同时显著提升计算效率，使研究人员能够在药物研发早期快速缩小候选分子范围，大幅降低实验筛选成本与时间投入。

| 能力 | 说明 |
|------|------|
| **大规模筛选** | 支持千万至亿级分子库的虚拟筛选 |
| **高精度对接** | 集成多种对接方法，提升结果可靠性 |
| **智能排序** | 深度学习打分模型辅助亲和力预测与重排 |
| **完整流程** | 覆盖前处理、对接、重排序与后处理全流程 |
| **高精度精筛** | 支持 MM/GBSA、FEP 等高精度计算方法 |

HelixVS 集成了多种先进的小分子对接方法，构建从粗筛到精筛的分层筛选体系。在筛选过程中，系统通过多种对接算法对分子进行初步筛选与精细对接，并结合深度学习打分模型对分子亲和力进行预测与重新排序，从而有效提高活性分子的命中率。相比传统单一对接方法，该多算法融合策略能够在保证高通量的同时提升筛选结果的可靠性与精度。


## 性能优势

在完整流程支持方面，HelixVS 提供覆盖前处理、对接筛选、结果重排序及后处理分析的端到端能力。系统支持分子与蛋白结构前处理、口袋定义与参数配置，并在筛选完成后提供多维度结果分析与过滤功能。用户可基于专家经验进行规则筛选，也可引入更高精度的计算方法，如 MM/GBSA、FEP 等，对候选分子进行进一步精筛与验证，从而获得更加可靠的候选结果。

在广泛用于评估虚拟筛选效果的 DUD-E 数据集（102 个靶点）上的测试结果表明，HelixVS 在精度与速度方面均优于主流虚拟筛选工具。相较业界常用的开源工具 Vina，HelixVS 平均能够多找到 159% 的活性分子，并在单机运行条件下实现超过 28 倍的速度提升。这意味着在相同计算资源条件下，HelixVS 可以在更短时间内发现更多潜在活性分子。

同时，相比基于深度学习的分子对接模型 KarmaDock 与商业软件 Glide，HelixVS 的平均命中率分别提升 70.3% 与 16.8%，在多类常见药物靶点上表现出更强的筛选能力，尤其在激酶、蛋白酶、核受体与离子通道等重要蛋白类别上表现更为突出。

| 方法 | EF0.1 | EF1 | 速度 |
|------|-------|-----|------|
| Vina | 17.1 | 10.0 | ~300 对/天 |
| Glide SP | 37.8 | 24.3 | ~2400 对/天 |
| KarmaDock | 26.0 | 15.8 | ~500万 对/天/GPU |
| **HelixVS** | **44.2** | **27.0** | **~8500 对/天** |

在帕金森病的重要靶点 COMT（Catechol-O-methyltransferase，儿茶酚-O-甲基转移酶）测试中，HelixVS 更是成为唯一能够成功找到活性分子的算法，EF0.1 高达 61.51，体现出其在真实药物研发场景中的实际应用价值与技术优势。

## 应用场景

HelixVS 虚拟筛选适用于多种药物研发场景，包括但不限于：

- **Hit Discovery**：从大型化合物库中发现新的潜在活性分子  
- **Lead Optimization**：对已知活性分子进行结构优化与筛选  
- **靶点研究**：探索蛋白口袋与配体相互作用模式  
- **AI+药物设计流程**：与分子生成或结构设计模型结合使用  

HelixVS 通过高精度、高通量与完整流程支持，为药物研发提供可靠且高效的虚拟筛选解决方案。
