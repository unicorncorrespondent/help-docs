# 设计结果解读

> 解读 Binder 设计结果的评分指标

Binder 设计任务完成后，系统会生成多个候选结果，并提供多维度评分指标用于评估其结合能力与结构可靠性。正确理解这些指标的含义，是筛选高质量 Binder 的关键步骤。

本页面将所有评分参数按实际使用功能进行分类，并提供清晰的判读逻辑与推荐阈值，帮助用户快速理解各项指标的作用，并高效筛选具有实验潜力的候选 Binder。

---

## 核心评分指标详解

### 结合能力评估

该类指标用于判断 Binder 与靶点之间是否能够形成稳定结合，是筛选候选结果的首要依据。

| 指标 | 符号 | 范围 | 说明 |
|------|------|------|------|
| **Helix Binder Score** | - | 0–1，越大越好 | 综合考虑结合稳定性与界面质量的总体评分 |
| **结合自由能** | ΔG | 越小（越负）越好 | kcal·mol⁻¹，预测结合稳定性 |
| **解离常数** | Kd | >0，越小越好 | 单位 M，预测亲和力 |

**筛选建议：**
- ΔG 与 Kd 建议同时参考，强结合通常表现为 ΔG 更负且 Kd 更小
- Helix Binder Score 可用于快速排序与初筛
- 若 ΔG 接近 0 或 Kd ≥ 1 μM，通常表示结合能力较弱



### 界面质量评估

该类指标用于评估 Binder 与靶点之间的界面构象是否可信，是判断结果可靠性的核心依据。

| 指标 | 全称 | 范围 | 说明 |
|------|------|------|------|
| **iPAE** | Interface Predicted Aligned Error | >0，越小越好 | Å，界面残基位置预测误差 |
| **ipLDDT_Target** | - | 0–100，越大越好 | 靶点界面区域结构可信度 |
| **ipLDDT_Binder** | - | 0–100，越大越好 | Binder 界面区域结构可信度 |
| **ipTM** | Interface PTM | 0–1，越大越好 | 链间相对位置与界面装配可信度 |

**筛选建议：**
- ipTM 是界面可靠性的核心指标，>0.8 表示高置信
- ipLDDT 用于判断界面残基构象是否可信
- iPAE 偏高可能提示界面预测误差较大
- 若 ipTM < 0.5，通常建议谨慎或排除

---

### 结构可靠性评估

该类指标用于判断预测结构整体是否稳定可靠，用于避免因模型不稳定导致误判。

| 指标 | 全称 | 范围 | 说明 |
|------|------|------|------|
| **Ranking Confidence** | - | 0–1，越高越好 | 综合评估构象整体可信度 |
| **pTM** | Predicted TM-score | 0–1 | 复合物整体结构预测准确性 |
| **pLDDT** | - | 0–100，越高越好 | 单残基局部结构预测可信度 |

**参考标准：**

| 指标 | 参考值 | 含义 |
|------|--------|------|
| pTM | >0.75 | 整体结构高度可信 |
| ipTM | >0.8 | 界面装配高度可信 |
| pLDDT | >90 | 局部结构高可信 |
| pLDDT | 70–90 | 可接受 |
| pLDDT | <70 | 结构可能不稳定 |

---

## 筛选

### 建议筛选流程

建议按以下顺序进行筛选：

**第一步：结合能力初筛**
- Helix Binder Score > 0.5
- ΔG < -5 kcal/mol
- Kd < 1 μM

不满足任一条件时，通常可优先排除。

**第二步：界面可靠性确认**
- ipTM > 0.8
- ipLDDT_Binder > 70
- iPAE 尽可能低

若 ipTM < 0.5，通常表示界面装配不可靠。

**第三步：整体结构复核**
- pTM > 0.75
- pLDDT > 70

若界面区域 pLDDT 明显偏低，应谨慎评估。

---

### 建议筛选阈值

| 指标 | 推荐阈值 | 含义 |
|------|----------|------|
| Helix Binder Score | > 0.5 | 综合表现良好 |
| ipTM | > 0.8 | 界面装配可靠 |
| ipLDDT_Binder | > 70 | 界面预测可信 |
| ΔG | < -5 kcal/mol | 结合稳定 |
| Kd | < 1 μM | 亲和力较强 |

---

### 高质量候选特征

综合以上指标，高质量 Binder 通常具备以下特征：

| 特征 | 说明 |
|------|------|
| 综合评分良好 | Helix Binder Score > 0.5 |
| 结合稳定 | ΔG 明显为负 |
| 亲和力较强 | Kd < 1 μM |
| 界面可靠 | ipTM > 0.8 |
| 界面可信 | ipLDDT > 70 |
| 结构合理 | pTM > 0.75 |

满足以上条件的候选结果，可优先下载并进入后续实验验证阶段。
