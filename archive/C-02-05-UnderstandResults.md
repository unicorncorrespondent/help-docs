# 理解设计结果

> 解读 Binder 设计结果的评分指标

Binder 设计任务完成后，系统会生成大量候选结果并提供丰富的评分指标。正确理解这些指标的含义是筛选优质设计结果的关键一步。本页面将详细介绍 HelixDesign-Binder 提供的各项评分维度，包括综合评分、结合自由能、界面置信度等核心指标，帮助您快速识别高质量的候选 Binder。


## 在线页面结果展示

### 结果概览

结果总览页展示所有候选 Binder 的**整体分布情况**，帮助您快速判断：

- 结果质量是否集中或分散
- 高质量候选的占比
- 哪些结果值得深入分析

### 评分维度

系统提供两种互补的分析视角：

| 分析维度 | 关注点 | 在线展示 |
|----------|--------|----------|
| **相互作用分析** | Binder 与靶点之间的结合质量 | 🖥️ |
| **复合物整体分析** | 整个复合物结构预测的可靠性 | 🖥️ |

### 评分分布图

#### 直方图 🖥️

展示所有候选 Binder 在某一指标上的分布情况：

- **横轴**：指标数值区间
- **纵轴**：落在对应区间内的 Binder 数量

#### 累积比例图 🖥️

评估不同结合质量阈值下的结果占比：

- 横轴：预设的结合强度阈值区间
- 纵轴：满足阈值的累积比例

### 表格功能 🖥️

- 点击表头可对指标进行**排序**
- 使用右上角**筛选功能**设置条件
- 点击「确认」后仅显示符合条件的候选项

## 核心评分指标 📊

### 结合相关指标

| 指标 | 全称 | 范围 | 说明 | 在线展示 | 下载文件 |
|------|------|------|------|----------|----------|
| **Helix Binder Score** | - | 0-1，越大越好 | 综合考虑结合稳定性、界面质量 | 🖥️ | 📁 |
| **结合自由能** | ΔG | 越小（越负）越好 | kcal·mol⁻¹，预测结合稳定性 | 🖥️ | 📁 |
| **解离常数** | Kd | >0，越小越好 | 单位 M，亲和力指标 | 🖥️ | 📁 |
| **iPAE** | - | >0，越小越好 | Å，界面预测误差 | 🖥️ | 📁 |
| **ipLDDT_Target** | - | 0-100，越大越好 | 靶点界面区域可信度 | 🖥️ | 📁 |
| **ipLDDT_Binder** | - | 0-100，越大越好 | Binder 界面区域可信度 | 🖥️ | 📁 |

### 结构可靠性指标

| 指标 | 全称 | 范围 | 说明 | 在线展示 | 下载文件 |
|------|------|------|------|----------|----------|
| **Ranking Confidence** | - | 0-1，越高越好 | 综合评估构象可信度 | 🖥️ | 📁 |
| **pTM** | Predicted TM-score | 0-1 | 全局结构准确性 | 🖥️ | 📁 |
| **ipTM** | Interface PTM | 0-1，>0.8 高置信度 | 亚基间相对位置 | 🖥️ | 📁 |
| **pLDDT** | - | 0-100，>90 高置信度 | 局部结构准确性 | 🖥️ | 📁 |

### pTM 与 ipTM 对比

| 指标 | 衡量对象 | 参考标准 |
|------|----------|----------|
| **pTM** | 整个复合物整体结构的预测准确性 | >0.75 高度可信，0.5-0.75 大致可信 |
| **ipTM** | 不同链之间相对位置的可信度 | >0.8 高置信度 |

### 指标详解与参考标准

| 指标 | 含义 | 参考标准 |
|------|------|----------|
| **结合自由能 (ΔG)** | Binder 与靶点结合的稳定性 | 越负越好，< -5 kcal/mol 为佳 |
| **解离常数 (Kd)** | 25°C 条件下预测的亲和力 | 越小越好，< 1 μM 亲和力较强 |
| **ipLDDT** | 界面区域残基构象预测可信度 | >90 极高，70-90 高，50-70 低，<50 极低 |
| **iPAE** | 界面处残基预测位置的平均误差 | 越小越好，数值越低表示构象预测越准确 |

## 筛选与下载 📁

### 建议筛选阈值

| 指标 | 推荐阈值 | 含义 |
|------|----------|------|
| Helix Binder Score | > 0.5 | 综合表现良好 |
| ipLDDT_Binder | > 70 | 界面预测可信 |
| ΔG | < -5 kcal/mol | 结合稳定 |
| Kd | < 1 μM | 亲和力较强 |

### 高质量候选特征

综合以上指标，高质量的 Binder 通常具备以下特征：

| 特征 | 说明 |
|------|------|
| 综合评分 | Helix Binder Score > 0.5 |
| 结合稳定性 | ΔG < -5 kcal/mol |
| 界面可靠性 | ipLDDT > 70 |
| 亲和力 | Kd < 1 μM |

## 下载设计结果

**获取结构文件和数据分析报告**

完成 Binder 设计结果的评估和筛选后，下一步就是将选定的候选结果下载到本地进行进一步的分析和使用。HelixDesign-Binder 提供完整的下载功能，包括预测的三维结构文件、评分数据汇总表以及可视化脚本。

### TLDR

- HelixDesign-Binder 的设计结果以 zip 压缩包的形式提供，包含进行完整分析所需的全部文件
- 下载包中包含用户输入信息记录文件（input.json）、所有设计结果的序列及评分汇总表（summary.csv）
- 每个候选 Binder 的详细预测文件包括：预测的三维结构文件（.cif 格式）、所有构象的评分结果、界面残基列表和 PyMOL 可视化脚本

### 下载内容

点击下载按钮获取设计结果，下载内容为 zip 压缩包。

### 文件结构

```
helixdesign_binder_results/
├── input.json                        # 用户输入信息记录
├── summary.csv                       # 所有设计结果的序列及评分汇总
└── results/
    ├── Binder_0000/
    │   ├── predicted_structure.cif   # 预测的三维结构文件
    │   ├── all_results.json         # 所有构象的评分结果
    │   ├── Interface.ic             # 相互作用残基列表
    │   └── InterfaceVis.pml        # PyMOL 可视化脚本
    ├── Binder_0001/
    │   ├── predicted_structure.cif
    │   ├── all_results.json
    │   ├── Interface.ic
    │   └── InterfaceVis.pml
    └── ...
```

### 文件说明

| 文件 | 说明 |
|------|------|
| **input.json** | 记录本次设计任务的输入参数与配置 |
| **summary.csv** | 汇总所有设计结果的序列信息及预测指标，便于整体对比与筛选分析 |
| **predicted_structure.cif** | 预测得到的三维结构文件 |
| **all_results.json** | 该设计结果下所有构象的评分信息 |
| **InterfaceVis.pml** | PyMOL 可视化脚本，用于展示对接界面残基 |
| **Interface.ic** | 对接界面残基列表 |

### PyMOL 可视化

**使用方法：**

1. 在 PyMOL 中打开对应的 `.cif` 结构文件
2. 加载 `InterfaceVis.pml` 脚本
3. 自动显示对接界面残基

**界面残基识别：**
- `Interface.ic` 文件列出界面处可能发生接触的残基对
- 可用于验证预测的结合模式

### 结果归档建议

| 文件 | 归档位置 | 说明 |
|------|----------|------|
| input.json | 项目文件夹 | 记录原始输入参数 |
| summary.csv | Excel 分析表 | 可直接导入 Excel 进行筛选 |
| cif 文件 | 结构数据库 | 按 Binder Index 命名归档 |
